Learning the MERN stack / October 2020

The MERN stack consists of 4 main technologies.

M - MongoDB - a noSQL database that uses JSON-style documents to store data from an application

E - Express - a node.js framework that offers robust routing, http helpers, content negotiation, and much more

R - React - a front-end component-based javascript framework for creating powerful interactive UIs

N - Node.js - a javascript back-end language that helps with generating dynamic page content

To learn the MERN stack, I developed an application called CookBook, which allows a user to create recipes, browse recipes, and favorite recipes. The application also features a built-in calculator for converting common units of volume, such as teaspoons, tablespoons, cups, pints, quarts, and gallons. There is also a larger page containing a chart with other common cooking conversions. The application is hosted on Heroku. Click to demo
