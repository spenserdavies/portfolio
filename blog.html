Learning the MERN stack

October 2020

The MERN stack consists of 4 main technologies:

M - MongoDB - a noSQL database that uses JSON-style documents to store data from an application

E - Express - a node.js framework that offers robust routing, http helpers, content negotiation, and much more

R - React - a front-end component-based javascript framework for creating powerful interactive UIs

N - Node.js - a javascript back-end language that helps with generating dynamic page content

   To learn the MERN stack, I developed an application called CookBook, which allows a user to create recipes, browse recipes, and favorite recipes. The application also features a built-in calculator for converting common units of volume, such as teaspoons, tablespoons, cups, pints, quarts, and gallons. There is also a larger page containing a chart with other common cooking conversions. The application is hosted on Heroku. Click to demo

Building a simple bank app

September 2020

   To keep myself practiced in Vue and C# / mySQL, I built out a simple banking application, called RealBank. This application had a few challenges, mostly with transfering funds, but once I got past them it didn't take to long to finish off. Users can edit transactions if they accidentally input a wrong amount or category, users can delete transactions (except for transfers), and users can transfer funds between accounts, which automatically creates two separate transactions. Accounts can also be closed, which acts as a transfer between the account you close and the account you move the money into, but then the account you are closing gets removed from the database. To view a demo video or use the application, navigate back to my portfolio page. The application uses Vue.js and boostrap on the front-end and uses C# and mySQL on the back. The application is hosted through a docker image on Heroku
